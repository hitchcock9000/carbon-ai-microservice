📊 Model Architecture & Comparison

🧠 LSTM (Deep Learning)

  • Architecture: Bidirectional LSTM + Attention
  • Layers: 128 → 64 units, Dropout 0.2
  • Framework: TensorFlow/Keras 2.16.2
  • Input: 24 timesteps × 15 features
  • MAE: 683.43 kWh
  • RMSE: 51,422.57 kWh
  • Training: 976 samples, 176 validation
  • Best for: 1-hour ahead forecasting

⚡ LightGBM (Gradient Boosting)

  • Algorithm: Gradient Boosted Decision Trees
  • Features: 14 (no lag features)
  • Training: 14,074,452 samples
  • Test: 3,025,569 samples
  • MAE: 656.18 kWh 🏆
  • RMSE: 63,555.27 kWh
  • R²: 0.000165 (vs 0 baseline)
  • Best for: 7-day ahead predictions

🎯 Key Findings & Insights

📈 Performance Comparison

  • Winner: LightGBM performs 4% better (656.18 vs 683.43 MAE)
  • LSTM Strength: 22% improvement over mean baseline (876.75 kWh)
  • Scale: LightGBM trained on 14M samples vs LSTM's 976 samples
  • Use Case: LSTM for real-time, LightGBM for planning

🔍 Technical Insights

  • Data Volume: LightGBM benefits from massive dataset (14M samples)
  • Feature Engineering: Both use temporal + weather + building features
  • LSTM Advantage: Captures sequential patterns with 24-hour lookback
  • LightGBM Advantage: No lag features = can predict far into future
